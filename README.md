# Installation & Development

### Minimum Requirements

- Node 18 or above

### Clone

Clone the repository using below command

```js
git clone https://github.com/Vasant-Mestry-11/altrata-calender-app.git
```

Run below commands to get started

```js
cd altrata-calender-app
npm install
npm run dev
```

### Test Cases

To run test case use below command
```js
npm run test
```
